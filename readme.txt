This is a simple handtracking and position tracking software. It can recognise 21 landmarks on any hands. This application uses mediapipe and opencv. 
This simple application can be imported as a module. There are a few application (1 completed and 1 in progress) within the class. The completed class
can return the position of the 21 landmarks in a list. The list data is modified to give us landmarks (x,y) cordinates. 

One application is to track movement of the hand.
